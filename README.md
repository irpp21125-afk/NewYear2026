# Discord Bot + Admin Panel (Python + HTML/CSS/JS)

Проект: Discord‑бот с экономикой/мини‑играми и отдельной веб‑панелью управления настройками/модерацией.

## Стек
- Bot: Python + `discord.py`
- Admin panel API: Python + `fastapi`
- UI: HTML/CSS/JS (статические файлы)
- Storage: SQLite (`aiosqlite`)

## Быстрый старт (Windows)

### 1) Создать виртуальное окружение и поставить зависимости
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### 2) Настроить переменные окружения
Скопируй `.env.example` в `.env` и заполни значения.

### 3) Инициализировать базу
```powershell
python -m bot.db
```

### 4) Запустить панель управления
```powershell
python -m panel.server
```
Открой в браузере: http://127.0.0.1:8000/

### 5) Запустить бота
В отдельном терминале:
```powershell
python -m bot.main
```

## Важные заметки по “картам remanga”
Этот проект хранит «карты» как сущности в **нашей** базе (id/название/владелец/источник). Автопроверка наличия карт на remanga.org будет добавлена после того, как будет понятно:
- как пользователь привязывает профиль (URL/ID)
- где на странице профиля отображаются «карты» и как их однозначно распознать

## Роли
- **Server owner** (владелец Discord‑сервера): полный доступ
- **Admin**: настройки/модерация/разбор нарушений
- **Moderator**: модерация игр и санкции по правилам

## Команды (MVP)
- `/balance [user]`
- `/daily`
- `/give user amount`
- `/coinflip amount`

Дальше расширим на ставки «картами», дуэли, правила, автобаны и журналирование.
